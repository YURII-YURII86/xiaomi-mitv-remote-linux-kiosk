# Support

This is an open-source project, not a hosted service.

## Before opening an issue

Please check:

- README quick start;
- docs folder;
- existing issues;
- latest release notes;
- `./scripts/smoke_test.sh` output.

## Good issue reports include

- repository name;
- version/commit;
- operating system;
- exact command run;
- expected behavior;
- actual behavior;
- logs with secrets and private identifiers removed.

## Production safety

If your issue involves a live kiosk, signage screen, Intel Stick, Raspberry Pi, or other appliance, mention whether the device is production-critical. Do not ask maintainers to guess whether a command is safe for your live device.
